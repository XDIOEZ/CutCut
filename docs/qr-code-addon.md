# 二维码扫描模块安装说明

二维码扫描是可独立安装和卸载的截图模块，需要轻截 `1.11.0` 或更高版本。将压缩包内的
`Modules\QrCode` 文件夹合并到轻截程序旁的 `Modules` 文件夹，然后等待轻截自动识别，
或在托盘菜单中选择“重新加载模块”。

安装后的文件为：

```text
Modules\QrCode\ScreenshotTool.QrCode.dll
Modules\QrCode\zxing.dll
Modules\QrCode\LICENSE-ZXING.NET.txt
```

安装后，像普通截图一样完整框选二维码，然后点击截图工具栏中的“二维码”。扫描在本机离线完成，
不会上传截图，也不会自动打开二维码中的网址或执行其中的内容。识别成功后截图界面会关闭，原始
二维码内容显示在选区旁边的小窗口中；结果可以继续编辑，也可以一键复制。若同一选区包含多个
可识别二维码，各项内容会分块显示。

可以在设置工作台的“插件模块”页面启用、禁用或永久删除二维码扫描。永久删除会移除整个
`Modules\QrCode` 文件夹；如需恢复，重新下载独立模块包并把同名文件夹放回程序旁即可。

二维码解码使用 Apache-2.0 许可的 ZXing.Net，许可文本随模块放在
`LICENSE-ZXING.NET.txt` 中。
