# 贴图悬浮窗模块

此安装包为轻截提供可独立装卸的贴图悬浮窗功能。解压后，将包内的 `Modules` 文件夹合并到
轻截程序目录；也可以直接把 `Modules\PinnedImage` 复制到程序旁的 `Modules` 中。

安装后重新开始一次截图，确认选区并完成标注，再点击工具栏中的“贴图”。当前最终选区会变成
置顶、无边框的悬浮图片，并关闭原截图编辑窗口。

- 左键拖动图片内部：移动贴图。
- 拖动四条边或四个角：默认保持原比例缩放。
- 按住 `Shift` 拖动边或角：自由改变宽高比例。
- 右键：打开“删除 / 复制 / 保存 / 编辑”菜单。
- “编辑”：关闭当前贴图，并把当前图片重新送回轻截编辑窗口。

复制和保存使用贴图原始像素；仅改变悬浮窗显示大小不会重采样或覆盖原图。编辑会把贴图当前像素
作为新的编辑底图，之后可继续添加、选择和修改新元素。

模块目录：

```text
Modules/
  PinnedImage/
    ScreenshotTool.PinnedImage.dll
```

贴图模块与包含 `IModuleImageHost` 契约的主程序版本配套使用。删除整个
`Modules\PinnedImage` 文件夹即可卸载；也可以在“插件模块”页禁用或永久删除。禁用、删除或替换
模块时，模块会关闭并释放它创建的全部贴图窗口。
