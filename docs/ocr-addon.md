# 本地 OCR 文字识别模块安装说明

本地 OCR 1.1.0 是可独立安装和卸载的截图模块，需要轻截 `1.11.0` 或更高版本。将压缩包内的
`Modules\Ocr` 文件夹合并到轻截程序旁的
`Modules` 文件夹，然后等待轻截自动识别，或在托盘菜单中选择“重新加载模块”。

安装后的入口文件为：

```text
Modules\Ocr\ScreenshotTool.Ocr.dll
```

安装后，像普通截图一样框选包含文字的区域，然后点击截图工具栏中的“OCR 本地”。轻截会在内存中
生成原图、高清放大、灰度对比度增强和 Otsu 二值化四种候选图，再调用 Windows 自带的离线 OCR
分别识别并选择信息最完整的结果，不会把图片上传到云端。识别成功后截图界面会关闭，结果显示在
选区旁边的小窗口中；文字可以继续编辑，也可以一键复制。

识别语言取决于 Windows 已安装的语言组件。若系统提示没有可用 OCR 语言，请在 Windows 的
“语言和区域”设置中为所需语言安装对应语言功能后重试。

可以在设置工作台的“插件模块”页面启用、禁用或永久删除本地 OCR。永久删除后，如需恢复，重新下载
独立模块包并把整个 `Modules\Ocr` 文件夹放回程序旁即可。

如果仍需要更高识别率，可另外下载独立的 PP-OCR Tiny 或 PP-OCR Small 模块。三种 OCR 可以同时安装，
但通常只启用其中一种，避免截图工具栏同时出现多个识别按钮。
