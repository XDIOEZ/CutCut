# 轻截录屏模块安装说明

录屏是可选模块，默认已预装在轻截完整包中，不会包含在独立基础包中。录屏模块 1.7.0 需要轻截基础程序 1.10.0 或更高版本；
升级录屏模块时也请更新基础程序，不能只把新版模块安装到旧版主程序中。版本匹配后，将压缩包里的
`Modules\ScreenRecording` 文件夹复制到轻截程序旁的 `Modules` 文件夹，然后重新启动轻截或在托盘菜单选择“重新加载模块”。卸载时删除整个：

```text
Modules\ScreenRecording\
```

安装后，先像普通截图一样框选同一显示器内的区域，截图工具栏会出现“录屏”按钮。录制支持：

- MP4 / H.264，30 或 60 FPS；
- 2、4、8、12、20 Mbps 五档视频码率；
- 根据视频码率和启用的音频实时估算 1 分钟、10 分钟、1 小时储存占用；
- 系统声音、麦克风，或两者混音；
- 录制从开始到结束都直接使用截图模式的同一条菜单栏，录屏模块不创建编辑按钮或专用控制窗；
- 模块程序集自带独立“录屏设置”分页，统一管理系统声音、麦克风、左键黄色圆圈、帧率、码率和储存估算；安装模块后页面自动出现，卸载后自动消失，点击“录屏”时直接采用已保存参数；
- 左键黄色半透明圆圈默认开启；开启时录制现场和最终 MP4 同步显示，按住左键期间持续跟随鼠标移动，松开后短暂保留，关闭时两处都不显示；
- 录制选区默认显示红色虚线范围提示，可在“录屏设置”中改为实线或不显示，该提示不会进入 MP4；
- 未选择编辑或绘图工具时鼠标穿透到被录制程序；录屏专属橙色“选择”按钮位于“矩形”左侧，开启后切换为高对比绿色“✓ 选择中”和白色强调边框，左键可单选、`Ctrl` 多选、框选、重叠轮选、移动和缩放元素；框选采用与截图模式一致的半透明蓝色填充与虚线边框，再次点击后恢复橙色“选择”和鼠标穿透状态；
- 全屏录屏编辑使用持久离屏画布进行局部合成，并缓存拖动期间的吸附参考，避免每次鼠标移动刷新整个透明全屏层或在清空旧位置时产生闪烁；
- 实时批注的菜单、命中、框选、`Ctrl` 多选、`Alt + 左键` 移动、八手柄缩放、`Ctrl` 固定步长拖动、元素吸附、删除和文字双击重编辑都由截图核心提供；
- 工具栏“吸附”按钮和快速双击 `Ctrl` 可切换当前会话的元素吸附；吸附距离与 `Ctrl` 拖动步长读取主程序“图片修改”设置；
- `Alt + 滚轮` 旋转、`Ctrl + 滚轮` 缩放，并支持粘贴图片或文字贴纸；
- 录制期间切换矩形、椭圆、箭头、画笔、文字和马赛克进行实时批注；
- 暂停、继续、撤销、清空和停止保存；
- 保存成功后在当前屏幕右下角显示提示，点击提示会打开资源管理器并选中对应 MP4 文件。

点击“停止并保存”且视频成功生成后，本次截图会话会直接结束，不会重新显示截图框。只有取消录屏或录屏保存失败时，宿主才恢复原截图框供用户继续操作。

录屏编码器需要 Windows Media Foundation 和 Microsoft Visual C++ x64 运行库。Windows N/KN 或
Windows Server 若缺少 Media Foundation，需要先安装相应的 Media Feature Pack / Media Foundation 功能。

录屏后端使用 MIT 许可的 ScreenRecorderLib，许可文本随模块包一同提供。
